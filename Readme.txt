ワイヤレスモーションセンサ C++ DLLの基本機能を使うためのGUIサンプル
2013/12/13

COMポートへの接続、切断、ステータス情報取得、電源OFFの機能を利用するためのサンプルです。

C++/MFC Visual Studio 2010にて作成。
（アプリケーションの種類はダイアログベースとしました）
（「Unicodeライブラリを使用する」のチェックを外しました）

作り方概要
・Visual StudioのウィザードでMFCアプリケーションを作成
・ヘッダファイル，libをリンク
  ライブラリのパス等も適宜設定します。
・Cmdクラスを宣言，初期化部に初期化処理を追加
・必要な機能を利用するためのボタン等を作成
・必要機能のコールバック関数を作成，初期化部で登録

※コールバック関数はメインと別のスレッドで呼ばれるため，
  ダイアログにアクセスする場合はメインスレッドにメッセージをPostする必要があります。
※サンプルの該当部分にコメントを挿入しています。
※サンプルではソースの変更はダイアログ（DllSampleCppDlg.h、DllSampleCppDlg.cpp）のみです。
※加速度は小型9軸（16G/1500dps）用の定数値で変換しています。
※計測時はプロダクトIDとセンサモジュールIDを指定してください。
